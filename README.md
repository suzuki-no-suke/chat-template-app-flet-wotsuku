## Getting started

  * Download source
  * Install python
  * Create venv & install & Run app

```
python -m venv .venv
.venv\bin\activate.bat
pip install -r requirements.txt
python main.py
```


## run test

Use `unittest` module

  * `python -m unittest discover`
